🚀 Start Contributing

Run the OpenPool node on your machine and earn credits by processing tasks from other users.

# Download and run the integrated node
git clone https://github.com/DP229/openpool.git
cd openpool && go build -o openpool ./cmd/integrated/
# Start the node
./openpool -port 9000 -http 8080 -workers 8
# With GPU support (coming soon)
./openpool -port 9000 -http 8080 -gpu

💡 Tips for Maximum Earnings

  • Keep the node running for maximum uptime credits
  • Use more workers (-workers flag) for concurrent task processing
  • Ensure stable internet connection for peer connectivity
  • GPU-equipped nodes earn higher credit rates
